API Beispiele

cURL
curl -s https://api.vatify.dev/v1/validate-vat \
  -H "Authorization: Bearer <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"vat_number":"DE811907980"}'
JavaScript (fetch)
const res = await fetch("https://api.vatify.dev/v1/validate-vat", {
  method: "POST",
  headers: {
    Authorization: "Bearer <API_KEY>",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({ vat_number: "DE811907980" }),
});
const data = await res.json();
cURL
curl -s https://api.vatify.dev/v1/calculate \
  -H "Authorization: Bearer <API_KEY>" \
  -H "Content-Type: application/json" \
  -d '{"country_code":"DE","rate_type":"standard","supply_date":"2025-09-01"}'
cURL
curl -s -H "Authorization: Bearer <API_KEY>" \
  https://api.vatify.dev/v1/rates/DE

Hinweis: Ersetze <API_KEY> mit deinem Schlüssel. Endpunkte laut MVP: /v1/validate-vat, /v1/calculate, /v1/rates/{country_code}.

FAQ

Gibt es ein Free Tier?

Ja, 100 Requests/Monat – ideal zum Testen.

Welche Länder werden unterstützt?

Alle EU-Länder. Die rates-Route liefert landesspezifische Steuersätze.

Wie authentifiziere ich?

Per Authorization: Bearer <API_KEY> Header.

Gibt es SDKs?

Bald. Aktuell cURL/Fetch Beispiele – reicht fürs MVP.
